# 投标资料库模板

这是一个简单版资料库结构，不要求你维护专业索引。

推荐结构：

```text
投标资料库\
  通用模板\
  历史投标材料\
```

## 通用模板

放通用版本的方案模板，例如：

```text
通用模板\
  技术方案模板.docx
  实施方案通用版.docx
  售后服务方案.docx
  培训方案.docx
  验收方案.docx
  风险管理方案.docx
```

## 历史投标材料

每个历史项目一个独立小文件夹。小文件夹里尽量同时放：

```text
历史投标材料\
  2024-某智慧园区项目\
    招标文件.pdf
    投标技术方案.docx
    澄清文件.pdf
  2025-某系统集成项目\
    招标文件.docx
    技术标.docx
```

这样 skill 可以先拿当前招标文件和历史项目里的招标文件做相似性判断，再决定是否复用对应的投标技术方案。

## 调用时怎么写

可以只给总目录：

```text
资料库在：E:\投标资料库
```

也可以分别给两个目录：

```text
通用模板在：E:\投标资料库\通用模板
历史投标材料在：E:\投标资料库\历史投标材料
```

不需要提前维护 `INDEX.md` 或 CSV。资料多了以后，如果想提升检索准确性，再补索引也不迟。
