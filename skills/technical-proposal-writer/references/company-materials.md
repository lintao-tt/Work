# Company Materials

Use this reference when the user wants the skill to reuse company-specific material across proposals.

## Suggested Source Library

Store reusable materials in a user-controlled directory, preferably outside temporary workspaces. A practical layout is:

```text
D:\Codex\BidProposals\_library\
  company-profile\
  technical-standard-wording\
  cases\
  certificates\
  product-docs\
  resumes\
  diagrams\
  templates\
```

If this directory is unavailable, ask for the current location of the user's library or keep placeholders.

## Material Rules

- Treat supplied company material as source evidence, not automatic truth for every tender.
- Reuse only cases and qualifications relevant to the tender's scope.
- Preserve official names, certificate numbers, project names, dates, and customer names exactly when provided.
- Do not fabricate missing names, dates, quantities, or performance results.
- When a case is useful but incomplete, write a gap note instead of padding the case.

## Template Handling

If the user provides a Word template:

- Preserve required headings, cover style, page setup, header/footer, catalog order, and table styles.
- Keep process files separate from the final Word document.
- If layout verification tools are available, render/inspect the final document before delivery.

If no template is provided, produce a clean technical draft and a response matrix. Avoid spending effort on visual polish until the content has passed compliance review.
